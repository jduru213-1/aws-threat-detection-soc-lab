# Scripts

- **`setup_splunk.py`** — Creates `aws_cloudtrail`, `aws_config`, `aws_vpcflow` in local Splunk. Requires `splunk-sdk` (`pip install splunk-sdk`). From repo root: `python ./scripts/setup_splunk.py`.
- **`knowledge_check.py`** — Optional lab quiz (CLI). Example: `python scripts/knowledge_check.py --level beginner`.
